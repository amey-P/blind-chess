# blind-chess
Terminal based chess interface using PGN notations.

# Dependencies:
- Python3.0 or above

- python-chess:
  ```bash
  pip install python-chess
  ```
# Usage:
Run ```main.py``` using any of the following:
```bash
python3 main.py
```
OR first convert into an executable
```bash
chmod +x main.py
```
then run,
```bash
./main.py
```
